① 为什么需要函数？

想象你要连续三次打印一个学生的成绩卡片——不用函数的话，你得把同样的代码复制三遍。如果格式要改，你要改三个地方。函数就是解决这个问题的。

❌ 不用函数：复制粘贴地狱
print("========")
print("姓名: 小明")
print("分数: 95")
print("========")

print("========")
print("姓名: 小红")
print("分数: 87")
print("========")

print("========")
print("姓名: 小刚")
print("分数: 92")
print("========")
# 如果格式要改，要改 6 行！
→
✅ 用函数：写一次，用多次
def print_card(name, score):
    print("========")
    print("姓名:", name)
    print("分数:", score)
    print("========")

print_card("小明", 95)
print_card("小红", 87)
print_card("小刚", 92)
# 格式要改？只改函数里 1 处！
🎮
生活中的函数类比

函数就像电视遥控器上的"一键静音"按钮——你不需要知道里面的电路是怎么连的，只要按一下按钮，就能执行一套预先设定好的操作。def 就是在"编程"这个遥控器，而调用函数就是"按按钮"。

📐
② def 关键字与语法格式

定义函数只需要记住一个关键字：def（define 的缩写，意为"定义"）。把它和函数名、冒号、缩进配合起来，就构成了函数的完整结构。

📋 函数语法结构
把鼠标悬停在每一行，看右侧说明
def 函数名():
第一行（函数头）：def 是关键字，告诉 Python"我要定义一个函数"。函数名 你自己起，命名规则和变量一样（字母/下划线开头，见名知意）。最后冒号不能省！
执行的代码A
函数体（缩进 4 格）：冒号下面、缩进的所有代码都属于这个函数。缩进是 Python 判断"哪些代码属于函数"的唯一方式，下面我们用可视化器来感受它。
执行的代码B
函数体可以有很多行，只要它们都保持同样的缩进级别，就都属于这个函数。
后续代码（不缩进）
一旦缩进"退回"到和 def 一样的级别，函数就结束了，后面的代码不再属于函数。
Python 
# 定义一个打招呼的函数
def say_hello():
    print("你好！")
    print("欢迎学习 Python 函数！")

# 定义不会执行任何输出，只是"登记"了函数
# 只有下面的调用，才会真正执行
say_hello()   # 输出：你好！ / 欢迎学习 Python 函数！

💡
缩进的黄金法则

Python 推荐用 4 个空格作为一级缩进。大多数编辑器（VS Code、PyCharm）按 Tab 键会自动插入 4 个空格。千万不要混用空格和 Tab，会导致 TabError！

记住这两句话

定义 = 告诉 Python "这个函数长什么样"，代码不执行，只是登记备案。
调用 = 真正让函数运行起来，每调用一次就执行一次函数体里的代码。

⑤ 执行流程步进器 — 跟着 Python 走一遍

Python 执行代码的顺序不是从上到下一行不跳的——调用函数时，它会"跳进"函数体执行，执行完再"跳回"原来的位置继续。点击"下一步"，亲眼看看每一行是怎么被执行的：

🔑
执行流程的核心规律：跳入 → 执行 → 跳回

每次遇到函数调用，Python 都会：① 暂停当前位置 → ② 跳入函数体执行 → ③ 执行完毕后跳回原来位置的下一行，继续执行。这个"跳入跳出"的过程就是函数调用的本质。

⑥ 综合示例

下面用一个稍完整的例子，把这节课学到的内容综合运用一下：

Python 
# 定义一个打印分割线的函数
def print_line():
    print("=" * 20)

# 定义一个打印欢迎信息的函数
def welcome():
    print_line()          # 函数里可以调用其他函数！
    print("欢迎来到课堂")
    print_line()

# 定义一个打印告别信息的函数
def goodbye():
    print_line()
    print("下节课见！")
    print_line()

# 主程序：调用函数
welcome()
print("今天我们学了函数的定义和调用")
print("记住：def 定义，括号调用！")
goodbye()

✨
函数可以调用函数！

在 welcome() 里调用了 print_line()，这完全没问题。函数之间可以互相调用，这是构建复杂程序的基础。但注意：被调用的函数必须在调用前已经定义好。

本节小结

学完这一节，你已经掌握了：

✓
函数是把一段代码"打包"起来的工具，用 def 关键字定义，用 函数名() 调用
✓
缩进是划定函数体范围的唯一方式——冒号下方 4 格缩进的代码都属于该函数
✓
定义不执行：def 语句只是"登记"函数，不会运行函数体里的代码
✓
调用才执行：写 函数名() 才会让函数体的代码真正运行，可以调用多次
✓
执行流程：遇到调用 → 跳入函数体 → 执行完毕 → 跳回原位继续
✓
函数内部可以再调用其他函数，但被调用的函数必须先定义
