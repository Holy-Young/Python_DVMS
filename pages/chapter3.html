① input() — 获取用户输入

到目前为止，我们写的程序都是"自说自话"的。但真正有用的程序需要跟用户"对话"—— 比如让用户输入名字、年龄或成绩。input() 就是完成这件事的函数。

Python 终端 — 运行效果演示
▶ # Python 显示提示，等待你在键盘上打字
请输入你的名字：小明 ← 用户输入后按 Enter

📌 基本语法

Python 
# 格式：变量名 = input("提示文字")
name = input("请输入你的名字：")
print("你好，" + name + "！欢迎学习 Python！")
⚠️
重要：input() 永远返回字符串！

不管用户输入了什么，input() 拿到的结果都是字符串（str）类型。如果你要把它当数字用，必须先用 int() 或 float() 转换，否则会报错。

🔢 输入数字时要先转类型

Python 
# ❌ 错误示范：直接用 input 的结果做数学运算
age_str = input("请输入年龄：")   # 用户输入 14，得到字符串 "14"
# print(age_str + 1)             # 报错！TypeError: 字符串不能 + 整数

# ✅ 正确做法：用 int() 转换成整数
age = int(input("请输入年龄："))  # 输入 14 → 整数 14
print("明年你", age + 1, "岁了") # 输出: 明年你 15 岁了

# ✅ 输入小数时用 float()
height = float(input("请输入身高(m)："))
print(f"你的身高是 {height} 米")

② if 判断 — 让程序"做决定"

现实中我们经常做判断："如果今天下雨，就带伞。" 程序里的 if 语句含义完全一样——如果某个条件成立（True），就执行对应的代码，否则跳过。

📐 基本语法格式

语法格式 
if 条件:
    # 条件为 True 时执行（注意：这里必须缩进！）
    执行的代码A
    执行的代码B

后续代码  # 不管条件真假都会执行

来看一个真实例子——判断用户能否进入游乐园（身高要求 ≥ 120 cm）：

Python 
height = int(input("请输入你的身高（厘米）："))

if height >= 120:
    print("✅ 欢迎进入！")
    print("祝你玩得开心~")

print("感谢光临！")  # 这行不管身高多少都会打印

🗺️ 执行流程图

开始
height ≥ 120 ？
True ✓
打印 "欢迎进入"
False ✗
（直接跳过）
打印 "感谢光临"（总会执行）
📏
⚠️ 缩进是 Python 的语法规则，不是可选项！
if 语句后面的代码块必须缩进（通常 4 个空格或 1 个 Tab）。 忘记缩进会直接报错 IndentationError！
❌ 错误写法
if score > 60: print("及格") # 没缩进 → IndentationError!
✅ 正确写法
if score > 60: print("及格") # 4格缩进 → 正常运行 ✓
🔄
③ if-else — "要么…要么…"

有时候我们需要："如果下雨就带伞，否则就带帽子。" 两条路必须走一条，这时候就用 if-else。

语法格式 
if 条件:
    # 条件为 True 时执行
    代码块 A
else:
    # 条件为 False 时执行
    代码块 B

🎯 例一：判断奇数或偶数

Python 
num = int(input("请输入一个整数："))

if num % 2 == 0:
    print(f"{num} 是偶数")
else:
    print(f"{num} 是奇数")

# 输入 8 → 8 是偶数
# 输入 7 → 7 是奇数

🎯 例二：判断是否及格

Python 
score = int(input("请输入你的成绩："))

if score >= 60:
    print("🎉 恭喜，及格了！")
else:
    print("😢 没及格，再加把劲！")

# 输入 75 → 🎉 恭喜，及格了！
# 输入 45 → 😢 没及格，再加把劲！

🗺️ if-else 执行流程

score ≥ 60 ？

打印 "恭喜及格"
False ✗
打印 "没及格"
结束
💡
记住：if 和 else 要"对齐"

else 和对应的 if 必须处于同一缩进级别（左边对齐），Python 才知道它们是一对。

🌿
④ if-elif-else — 多重条件判断

如果有三种或更多情况要分别处理，就用 if-elif-else。 elif 是 "else if" 的缩写。Python 从上往下依次检查， 一旦某个条件成立就执行对应代码块，然后直接跳出整个判断，后面的 elif/else 不再检查。

语法格式 
if 条件1:
    执行代码A      # 条件1 成立
elif 条件2:
    执行代码B      # 条件1 不成立，条件2 成立
elif 条件3:
    执行代码C      # 前两个都不成立，条件3 成立
else:
    执行代码D      # 以上全不成立时，兜底执行

score = int(input("请输入成绩（0-100）："))

if score >= 90:

⑤ 三种形式对比总结

来总结一下三种写法各自的使用场景：

① 只有 if
条件成立时做某件事，不成立就什么都不做。适合"可做可不做"的场景。 if score == 100: print("满分！")
② if-else
两种情况必选其一，非 A 即 B。适合"非此即彼"的场景。 if age >= 18: print("成年") else: print("未成年")
③ if-elif-else
三种及以上情况，从上往下依次判断，满足就停。适合分类、分级场景。 if score >= 90: print("优秀") elif score >= 60: print("及格") else: print("不及格")
🪆
⑥ 嵌套 if — if 里面再写 if

有时候一个判断通过后，还需要继续判断，这就是嵌套 if——把 if 写在另一个 if 的代码块里面。

外层 if：先判断年龄
如果 age >= 16 → 通过年龄验证，进入下一步
内层 if-else：再判断是否有票
如果 has_ticket == "yes" → 打印 "✅ 欢迎入场！"
否则 → 打印 "❌ 请先购票"
Python 
age = int(input("请输入年龄："))
has_ticket = input("有票吗？(yes/no)：")

if age >= 16:
    if has_ticket == "yes":
        print("✅ 欢迎入场！")
    else:
        print("❌ 请先购票再来")
else:
    print("❌ 未满 16 岁，不能入场")

# 年龄 18，有票 yes → ✅ 欢迎入场！
# 年龄 18，有票 no  → ❌ 请先购票再来
# 年龄 12，有票 yes → ❌ 未满 16 岁，不能入场
💡
嵌套 if 要注意缩进层级

每嵌套一层就多缩进 4 格。嵌套太多层会让代码很难读，一般不超过 3 层。 很多嵌套情况都可以用 and 合并条件来简化，比如把上面的代码改写成 if age >= 16 and has_ticket == "yes"。

本节要点

🎯 本节重点回顾

1. input() 语句

获取用户键盘输入，返回值永远是字符串；需要数字时必须用 int() 或 float() 转换。

2. if 语句

条件为 True 时执行缩进的代码块。冒号和缩进缺一不可！

3. if-else 语句

两条路选一条，条件为 True 走 if，条件为 False 走 else。

4. if-elif-else 语句

多个条件从上往下依次判断，匹配到第一个就执行并跳出，else 兜底处理。

5. 缩进规则

Python 用缩进（4 个空格）划分代码块，不是装饰，是语法的一部分！忘记缩进 = 报错。
