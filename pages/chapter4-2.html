① 什么是参数？

上节课我们定义了函数，但函数每次执行做的事是完全一样的。如果想让函数处理不同的数据，就需要用到参数——告诉函数"这次要处理什么"。

🧋
用奶茶店来理解参数
函数就像一台奶茶机器，参数就是你下单时告诉它的信息。
"我要一杯大杯、少糖、去冰的珍珠奶茶！"
没有参数，机器每次只会做同一种奶茶；有了参数，才能按你的要求定制！

Python 中有三种传参方式，点击下面的标签切换学习：

位置参数是最基础的传参方式——调用时按照定义的顺序依次传入，第一个值给第一个参数，第二个值给第二个参数，以此类推。
调用：
greet("小明", 14)
→
定义：
def greet(name, age)
🟣 "小明" → name 🟣 14 → age
Python 
def greet(name, age):          # 定义两个位置参数
    print(f"你好，{name}！")
    print(f"你今年 {age} 岁。")

greet("小明", 14)   # "小明" 给 name，14 给 age
greet("小红", 13)   # "小红" 给 name，13 给 age

# 输出：
# 你好，小明！
# 你今年 14 岁。
# 你好，小红！
# 你今年 13 岁。
⚠️
顺序很重要！

位置参数必须按顺序传入，多传或少传都会报错。greet(14, "小明") 和 greet("小明", 14) 是不同的！

默认参数就是在定义函数时，给参数提前设好一个值。调用时如果没有传这个参数，就用默认值；如果传了，就用你传的值。
定义（sweet 有默认值）：
def order(size, sweet="标准")
order("大杯")
→
size="大杯", sweet="标准"（用默认值）
order("大杯", "少糖")
→
size="大杯", sweet="少糖"（覆盖默认值）
Python 
def order(size, sweet="标准"):   # sweet 有默认值
    print(f"正在制作：{size}，甜度：{sweet}")

order("大杯")             # 不传 sweet，用默认的"标准"
order("中杯", "少糖")     # 传了 sweet，用"少糖"
order("小杯", "无糖")     # 传了 sweet，用"无糖"

# 输出：
# 正在制作：大杯，甜度：标准
# 正在制作：中杯，甜度：少糖
# 正在制作：小杯，甜度：无糖
💡
默认参数要放在最后面

定义函数时，有默认值的参数必须放在没有默认值的参数后面。写成 def order(sweet="标准", size) 会报错！

关键字参数是在调用时，直接用 参数名=值 的方式传入。这样就不用管顺序了，想怎么传就怎么传——就像直接"点名"告诉函数每个参数是什么。
两种调用方式，结果完全一样：
order("大杯", "少糖")
← 位置参数（靠顺序）
order(sweet="少糖", size="大杯")
← 关键字参数（靠名字，顺序随意）
Python 
def make_tea(size, sweet, temperature):
    print(f"{size} | {sweet} | {temperature}")

# 位置参数（必须按顺序）
make_tea("大杯", "少糖", "去冰")

# 关键字参数（顺序随意，靠参数名对应）
make_tea(temperature="热", size="中杯", sweet="全糖")

# 混用也可以：位置参数在前，关键字参数在后
make_tea("大杯", temperature="正常冰", sweet="少糖")

# 三行输出一样清晰，各自对应正确的参数
💡
什么时候用关键字参数？

当函数参数很多、容易记混顺序的时候，用关键字参数可以让代码更清晰、更不容易出错！

📤
② 什么是返回值？

参数解决了"怎么给函数数据"的问题。那函数算完之后，结果怎么告诉我们？答案就是 return——让函数把结果"吐"回来。

🧮
用计算器来理解 return
你按下计算器的 3 + 5 =，它把 8 显示出来还给你——这就是返回值！
函数执行完计算后，用 return 把结果"交还"给调用者，调用者可以用一个变量接住它，继续使用。

📌 有无 return 的对比

😅 只 print，不 return
结果只会显示在屏幕上，你没办法拿到这个结果继续用。 def add(a, b): print(a + b) # 只是打印 result = add(3, 5) # 打印了8 print(result) # 输出: None ！
✅ 用 return 返回结果
结果被"吐"回来，你可以用变量接住，想怎么用都行！ def add(a, b): return a + b # 返回结果 result = add(3, 5) # 接住8 print(result) # 输出: 8 print(result * 2) # 还能继续用！

📝 代码示例：有返回值的函数

Python 
# 计算两个数的和，并返回结果
def add(a, b):
    result = a + b
    return result     # 把结果返回给调用者

# 接住返回值，才能继续使用
sum1 = add(3, 5)
print(sum1)           # 8

# 返回值可以直接参与运算
print(add(10, 20) + 5)  # 35

# 返回值可以作为另一个函数的参数
print(f"结果是：{add(100, 200)}")   # 结果是：300

⚡ return 会立刻结束函数

执行到 return 那一行，函数立刻结束，后面的代码都不会执行：

Python 
def check_age(age):
    if age >= 18:
        return "成年人"    # 执行到这里，函数立刻结束
    return "未成年人"      # 上面 return 了就轮不到这里

result = check_age(20)
print(result)   # 成年人

result = check_age(15)
print(result)   # 未成年人

🎁 返回多个值

Python 允许一个函数同时 return 多个值，用逗号隔开就行。Python 会自动把它们打包在一起，接收时可以用多个变量一起接住：

🔍 返回多个值的过程
函数内部
return 10, 3
→
自动打包
(10, 3)
→
两个变量接住
quotient = 10
remainder = 3
Python 
# 同时返回商和余数
def divide(a, b):
    quotient  = a // b   # 整除，得到商
    remainder = a % b    # 取余，得到余数
    return quotient, remainder   # 同时返回两个值

# 用两个变量分别接住
q, r = divide(23, 5)
print(f"23 ÷ 5 = {q} 余 {r}")   # 23 ÷ 5 = 4 余 3

# 也可以返回最大值和最小值
def max_min(a, b, c):
    return max(a, b, c), min(a, b, c)

big, small = max_min(8, 3, 15)
print(f"最大值：{big}，最小值：{small}")  # 最大值：15，最小值：3
💡
不写 return 会怎样？

函数执行完毕后，默认返回一个特殊的值 None，意思是"什么都没有"。所以如果你需要使用函数的计算结果，记得加上 return！

🔭
③ 变量的"生命范围"（了解即可）

有一个小知识点简单了解一下：函数里面定义的变量，出了函数就消失了，在函数外面用不了。你不需要担心函数里的变量名和外面"撞名"。

🌍 全局范围（整个程序都能用）
score = 100 ← 在函数外定义，整个程序都能读取
📦 函数内部
result = score + 10 ← 可以读取外面的 score
temp = result * 2 ← 只在函数内有效
🚫 函数外面访问 result 或 temp → 报错！NameError（它们出了函数就不存在了）
Python 
score = 100   # 全局变量

def calculate():
    result = score + 10   # 能读取全局的 score
    return result

print(calculate())  # 110，正常

# print(result)     # ❌ 报错！result 只存在于函数内部
                    # 出了函数，它就"消失"了
🎯
记住一句话就够了

函数里面的变量，出了函数门就不存在。所以函数里的"草稿纸"不用担心影响外面——这也是函数能帮我们管理代码的原因之一。

🎮
④ 综合练习

来看三个综合运用参数和返回值的实用小函数：

📐 练习一：矩形面积计算器
这个函数接收长和宽，返回面积。可以反复调用，处理不同的矩形。
Python 
def rectangle_area(length, width):
    """计算矩形面积"""
    area = length * width
    return area

# 调用并接住返回值
area1 = rectangle_area(5, 3)
area2 = rectangle_area(10, 4)
area3 = rectangle_area(2.5, 8)

print(f"5×3 的矩形面积：{area1}")    # 15
print(f"10×4 的矩形面积：{area2}")   # 40
print(f"2.5×8 的矩形面积：{area3}")  # 20.0

# 返回值可以继续运算
total = area1 + area2 + area3
print(f"三个矩形总面积：{total}")     # 75.0
💬 练习二：问候语生成器（带默认参数）
用默认参数让函数更灵活——大多数情况下用默认问候语，有特殊需求再传入自定义的。
Python 
def greet(name, greeting="你好", punctuation="！"):
    """生成问候语，有默认参数"""
    message = f"{greeting}，{name}{punctuation}"
    return message

# 只传名字，其余用默认值
print(greet("小明"))                         # 你好，小明！

# 传入自定义问候语
print(greet("老师", greeting="早上好"))       # 早上好，老师！

# 关键字参数，顺序随意
print(greet(punctuation="~", name="小红", greeting="嗨")) # 嗨，小红~

# 返回值存到变量里，继续使用
msg = greet("全班同学", greeting="大家好")
print(msg)
print(f"[公告] {msg}")
📊 练习三：同时返回最大值和最小值
一次函数调用，返回多个结果——省去了调用两次函数的麻烦。
Python 
def get_max_min(scores):
    """传入一个成绩列表，同时返回最高分和最低分"""
    highest = max(scores)  # 用 Python 内置的 max()
    lowest  = min(scores)  # 用 Python 内置的 min()
    return highest, lowest  # 同时返回两个值

# 准备成绩列表
class_scores = [85, 92, 78, 96, 63, 88, 71]

# 用两个变量接住返回的两个值
best, worst = get_max_min(class_scores)
print(f"全班最高分：{best} 分")    # 96
print(f"全班最低分：{worst} 分")   # 63

# 还可以继续计算分差
gap = best - worst
print(f"最高分和最低分相差 {gap} 分")  # 33

本节小结

学完这一节，你应该掌握：

✓
位置参数：按顺序传，第一个给第一个，第二个给第二个
✓
默认参数：定义时设好默认值，调用时可以不传；有默认值的参数要放在最后
✓
关键字参数：用 参数名=值 传入，顺序随意，代码更清晰
✓
return：把函数的计算结果"吐"回给调用者，调用者用变量接住后可以继续使用
✓
return 立刻结束函数：执行到 return，函数立刻停止，后面的代码不会执行
✓
返回多个值：return a, b，用两个变量 x, y = func() 分别接住
✓
变量作用域：函数里的变量出了函数就消失，不会和外面的变量相互干扰
