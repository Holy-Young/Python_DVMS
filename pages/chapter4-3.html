① lambda 匿名函数

用 def 定义的函数有名字，有时候我们只需要一个"用完即扔"的小函数——比如临时告诉 sorted() 按什么规则排序。这时候写一个完整的 def 函数就太麻烦了，lambda 应运而生。

💡
一个比喻

def 就像在餐厅点一道正式菜，有名字有菜单；lambda 就像临时跟厨师说"给我炒个蛋就行"——说完就用，用完就算，不需要给它起个名字。

📐 语法格式对比

🐢 def 写法（完整函数）
# 定义一个函数，计算平方
def square(x):
    return x * x

# 调用
print(square(5))   # 25
→
⚡ lambda 写法（一行搞定）
# 用 lambda 表达同样的逻辑
square = lambda x: x * x

# 调用方式完全一样
print(square(5))   # 25

📌 语法规则

语法格式 
lambda 参数1, 参数2, ... : 表达式
#  ↑ 关键字    ↑ 参数列表   ↑ 直接是返回值的表达式（不用写 return）

# 例子
add   = lambda a, b: a + b          # 两数相加
greet = lambda name: "你好，" + name  # 拼接字符串
is_even = lambda n: n % 2 == 0     # 判断是否为偶数

print(add(3, 5))          # 8
print(greet("小明"))       # 你好，小明
print(is_even(4))          # True
⚠️
lambda 只能写一个表达式

lambda 的函数体只能是一行表达式，不能写 if 语句、for 循环或多条语句。如果逻辑复杂，就老老实实用 def。

② 函数作为参数——sorted、map、filter

Python 中，函数可以像普通数据一样被当作参数传给另一个函数。这样的函数叫做"高阶函数"。sorted()、map()、filter() 是三个最常用的内置高阶函数，和 lambda 配合起来威力巨大。

🎮 高阶函数模拟器——点击看执行过程

sorted(列表, key=函数)：对列表排序，key 指定排序依据。每个元素都会被 key 函数处理一次，再按处理结果排序。

Python 
students = [
    {"name": "小明", "score": 85},
    {"name": "小红", "score": 92},
    {"name": "小华", "score": 78},
]

# 按成绩从高到低排序
result = sorted(students, key=lambda s: s["score"], reverse=True)

for s in result:
    print(f"{s['name']}: {s['score']} 分")

# 输出：
# 小红: 92 分
# 小明: 85 分
# 小华: 78 分

map(函数, 列表)：把列表的每个元素都传给函数处理一遍，收集所有结果。就像流水线上的加工机器，把所有原料一个一个转化。

Python 
scores = [85, 92, 78, 96, 63]

# 给每个分数加 5 分
new_scores = list(map(lambda x: x + 5, scores))
print(new_scores)   # [90, 97, 83, 101, 68]

# 把每个数字转成字符串
str_scores = list(map(str, scores))
print(str_scores)   # ['85', '92', '78', '96', '63']

filter(函数, 列表)：对列表里的每个元素判断"留还是不留"——函数返回 True 就保留，返回 False 就丢掉。

Python 
scores = [85, 42, 91, 58, 76, 30, 88]

# 只保留及格（>=60）的成绩
passed = list(filter(lambda x: x >= 60, scores))
print(passed)   # [85, 91, 76, 88]

# 只保留偶数
nums = [1, 2, 3, 4, 5, 6, 7, 8]
evens = list(filter(lambda x: x % 2 == 0, nums))
print(evens)    # [2, 4, 6, 8]

🔑
map 和 filter 返回的是"懒对象"，要用 list() 包一下

map() 和 filter() 返回的不是列表，而是一个特殊的"懒对象"，需要套一层 list() 才能转换成列表使用。直接 print(map(...)) 会看到类似 <map object at 0x...> 的奇怪输出。

📦
③ 可变参数 *args

有时候我们事先不知道要传多少个参数——比如求若干个数的总和，可能传 2 个，也可能传 10 个。这时候用 *args 来接收任意数量的参数，Python 会把它们全部打包成一个元组传进来。

Python 
def my_sum(*args):
    print(args)         # args 是一个元组
    total = 0
    for num in args:    # 遍历元组
        total += num
    return total

print(my_sum(1, 2))          # args=(1, 2)  → 3
print(my_sum(1, 2, 3))       # args=(1, 2, 3) → 6
print(my_sum(10, 20, 30, 40)) # args=(10, 20, 30, 40) → 100

🔍 *args 和普通参数混用

Python 
# 普通参数在前，*args 在后
def greeting(prefix, *names):
    for name in names:
        print(f"{prefix}，{name}！")

greeting("你好", "小明", "小红", "小华")
# 输出：
# 你好，小明！
# 你好，小红！
# 你好，小华！

💡
星号 * 的含义

参数名前面加 * 就代表"收集剩余所有参数"的意思。名字不一定要叫 args，叫 *numbers、*items 都行，但约定俗成大家都写 *args。

📊 练习一：成绩分析系统（综合运用内置函数）
Python
scores = [85, 42, 91, 58, 76, 30, 88, 95, 67, 73]

# 基本统计（内置函数一步搞定）
print(f"人数：{len(scores)} 人")
print(f"最高分：{max(scores)} 分")
print(f"最低分：{min(scores)} 分")
print(f"总分：{sum(scores)} 分")
print(f"平均分：{round(sum(scores) / len(scores), 1)} 分")

# 过滤及格成绩
passed = list(filter(lambda x: x >= 60, scores))
print(f"\n及格人数：{len(passed)} / {len(scores)}")
print(f"及格成绩：{passed}")

# 从高到低排序
ranked = sorted(scores, reverse=True)
print(f"\n排名榜：{ranked}")
🏷️ 练习二：学生排行榜（lambda + sorted）
Python
students = [
    {"name": "小明", "score": 85, "age": 14},
    {"name": "小红", "score": 92, "age": 13},
    {"name": "小华", "score": 85, "age": 15},  # 同分用年龄排
    {"name": "小李", "score": 78, "age": 14},
]

# 先按分数降序，分数相同按年龄升序
ranked = sorted(
    students,
    key=lambda s: (-s["score"], s["age"])
)

print("=== 排行榜 ===")
for i, s in enumerate(ranked, 1):
    print(f"第{i}名：{s['name']}  {s['score']}分  {s['age']}岁")
🧮 练习三：灵活求和（*args + 内置函数）
Python
def analyze(*scores):
    """接收任意个成绩，返回统计结果"""
    if not scores:
        return "没有传入任何成绩"
    return {
        "count":   len(scores),
        "highest": max(scores),
        "lowest":  min(scores),
        "total":   sum(scores),
        "average": round(sum(scores) / len(scores), 1)
    }

# 传入不同数量的参数
r1 = analyze(85, 92, 78)
r2 = analyze(95, 88, 76, 63, 91, 82)

print(r1)  # {'count': 3, 'highest': 92, ...}
print(r2)  # {'count': 6, 'highest': 95, ...}

本节小结

学完这一节，你已经掌握了：

✓
lambda：lambda 参数: 表达式，一行搞定简单函数，表达式的值就是返回值，不用写 return
✓
sorted(list, key=lambda)：按指定规则排序，key 接收一个函数，对每个元素"取值"后再排
✓
map(函数, 列表)：把列表中每个元素都用函数处理一遍，需要 list() 包裹后使用
✓
filter(函数, 列表)：保留让函数返回 True 的元素，丢弃 False 的元素，同样需要 list()
✓
*args：函数参数前加 *，接收任意数量的参数，函数体内是一个元组
✓
常用内置函数：len()、max()、min()、sum()、abs()、round()，无需 import 直接调用
