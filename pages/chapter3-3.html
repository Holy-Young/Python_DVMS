① for 循环 vs while 循环

学完 while 循环，你已经知道怎么让程序"重复做事"了。那么 for 循环又是什么？和 while 有什么不同？

💡
一句话记住区别

while 是"条件驱动"——只要条件成立就一直转，不知道要跑几次；for 是"序列驱动"——从头到尾把序列里的每个元素取出来用一遍，自然结束。

🔄 while 循环
条件驱动

适合"不知道要循环几次，满足某个条件才停"的场景，比如猜数字游戏、反复等待用户输入。

需要自己维护计数变量，容易忘记 i += 1 造成死循环。

# 打印 0 到 4 i = 0 while i < 5: print(i) i += 1 # 别忘了！
🔁 for 循环
序列驱动

适合"知道要遍历什么内容"的场景，比如处理列表里的每个元素、依次打印字符串中的每个字。

不需要手动维护计数变量，更简洁，不容易出错，是 Python 中最常用的循环。

# 打印 0 到 4（更简洁） for i in range(5): print(i) # 自动结束，不会死循环
✅ 怎么选？
· 遍历字符串、列表、数字序列 → 用 for
· 不知道循环几次、依赖条件退出 → 用 while
· 简单记：能数清楚次数就用 for，数不清楚就用 while
🔁
② for 基本语法与执行流程

📐 基本语法格式

语法格式 
for 变量 in 序列:
    # 每次从序列中取出一个元素，赋值给"变量"，然后执行这里的代码
    执行的代码A
    执行的代码B

后续代码  # 序列遍历完后，才执行这里

🗺️ 执行流程图

开始
↓

↓

↩ 回去继续取
✗ 没有了

💡 for 循环会自动把序列中的元素一个一个取完，不需要手动计数，取完就自动停止

📝 遍历字符串——逐个字符

字符串本质上是一个字符序列，for 循环可以把它一个字一个字地取出来：

Python 
# 遍历字符串，逐个打印每个字符
word = "Python"
for char in word:
    print(char)

# 输出：
# P
# y
# t
# h
# o
# n
Python 
# 统计字符串中某个字符出现的次数
sentence = "banana"
count = 0
for char in sentence:
    if char == "a":
        count += 1
print(f"字母 a 出现了 {count} 次")
# 输出：字母 a 出现了 3 次

📋 遍历列表——逐个元素

列表（list）是存放多个数据的容器，for 循环是处理列表最常用的方式：

Python 
# 遍历列表，给每位同学打招呼
students = ["小明", "小红", "小华", "小李"]
for name in students:
    print(f"你好，{name}！")

# 输出：
# 你好，小明！
# 你好，小红！
# 你好，小华！
# 你好，小李！
Python 
# 计算列表中所有成绩的总和与平均分
scores = [85, 92, 78, 96, 88]
total = 0

for score in scores:
    total += score

average = total / len(scores)
print(f"总分：{total}")
print(f"平均分：{average}")
# 输出：
# 总分：439
# 平均分：87.8

for 循环的循环变量

for 后面的那个变量（如 name、char、score）叫做循环变量，每次循环它都会自动更新为序列的下一个元素，不需要手动修改它！

③ range() 函数——生成数字序列

range() 函数可以生成一个整数序列，配合 for 循环使用，就能实现"循环指定次数"。 这是最常见的 for 循环写法！

📦 三种用法

Python 
# 用法一：range(stop) — 从 0 开始，到 stop-1 结束
for i in range(5):
    print(i, end=" ")
# 输出：0 1 2 3 4

# 用法二：range(start, stop) — 自定义起点
for i in range(1, 6):
    print(i, end=" ")
# 输出：1 2 3 4 5

# 用法三：range(start, stop, step) — 自定义步长
for i in range(0, 10, 2):   # 步长为 2，每次跳 2 个
    print(i, end=" ")
# 输出：0 2 4 6 8

# 倒序（步长为负数）
for i in range(5, 0, -1):
    print(i, end=" ")
# 输出：5 4 3 2 1
🔑
记住"左闭右开"原则

range(start, stop) 包含 start，但不包含 stop。比如 range(1, 6) 生成的是 1、2、3、4、5，没有 6。这和数学里的"左闭右开区间 [1, 6)"是一样的道理！

④ 嵌套 for 循环

for 循环里面可以再嵌套一个 for 循环。外层每循环一次，内层就完整地跑完一遍。 这在打印图案、处理表格数据时非常常用。

📐 基本结构

语法格式 
for i in range(外层次数):      # 外层：控制"行"
    for j in range(内层次数):  # 内层：控制"列"
        执行操作
    # 每行结束后可以换行

⭐ 打印星号图案

Python 
# 打印 4 行 × 5 列的矩形
for i in range(4):        # 外层：4 行
    for j in range(5):    # 内层：每行 5 个
        print("★", end=" ")
    print()               # 每行结束换行
# 输出：
# ★ ★ ★ ★ ★
# ★ ★ ★ ★ ★
# ★ ★ ★ ★ ★
# ★ ★ ★ ★ ★

# 打印直角三角形（内层次数随外层变化）
for i in range(1, 6):     # i = 1, 2, 3, 4, 5
    for j in range(i):    # 第 i 行打印 i 个星
        print("★", end=" ")
    print()
# 输出：
# ★
# ★ ★
# ★ ★ ★
# ★ ★ ★ ★
# ★ ★ ★ ★ ★

🏆 九九乘法表（for 版本）

九九乘法表是嵌套循环最经典的例子，用 for 写起来比 while 更简洁：

Python 
for i in range(1, 10):           # 外层：被乘数 1~9
    for j in range(1, i + 1):    # 内层：乘数 1~i
        print(f"{j}×{i}={i*j}", end="\t")
    print()                      # 每行结束换行

⚡
嵌套循环的执行次数 = 外层次数 × 内层次数

比如外层跑 4 次、内层跑 5 次，一共执行 4 × 5 = 20 次。层数越多，执行次数增长越快，一般不超过三层嵌套。

🛑
⑤ break 与 continue（for 版）

在 while 循环里学过的 break 和 continue，在 for 循环中的用法和效果完全一样！

🛑 break — 立刻退出
遇到 break，整个 for 循环立刻结束，跳到循环后面继续执行。常用于"找到目标就停"。 for i in range(1, 10): if i == 5: break # 到 5 就停 print(i, end=" ") # 输出：1 2 3 4
⏭️ continue — 跳过本次
遇到 continue，跳过本次剩余代码，直接进入下一次循环。常用于"过滤不需要的元素"。 for i in range(1, 8): if i % 2 == 0: continue # 跳过偶数 print(i, end=" ") # 输出：1 3 5 7

🔍 break 实战：查找列表中的目标

Python 
# 在学生名单中查找"小华"，找到就停止
students = ["小明", "小红", "小华", "小李", "小张"]
target = "小华"

for name in students:
    print(f"检查：{name}")
    if name == target:
        print(f"✅ 找到了！{target} 在名单中")
        break   # 找到就立刻停，不再检查后面的人
else:
    # for 循环正常结束（没有被 break 打断）时执行
    print(f"❌ {target} 不在名单中")

# 输出：
# 检查：小明
# 检查：小红
# 检查：小华
# ✅ 找到了！小华 在名单中
🌟
for...else 结构（Python 特有）

for 循环后面可以加 else——当循环正常走完（没有被 break 打断）时，才执行 else 里的代码。这是 Python 特有的语法，非常适合"查找"场景。

🍬 continue 实战：跳过不需要的元素

Python 
# 只打印列表中的正数，跳过负数和零
numbers = [3, -1, 7, 0, -5, 8, 2]

print("正数有：", end="")
for num in numbers:
    if num <= 0:
        continue    # 不是正数，跳过
    print(num, end=" ")
# 输出：正数有：3 7 8 2

# 过滤掉不及格的成绩，只打印及格的
scores = [85, 42, 91, 58, 76, 30, 88]
print("及格成绩：", end="")
for score in scores:
    if score < 60:
        continue
    print(score, end=" ")
# 输出：及格成绩：85 91 76 88

本节要点

学完这一节，来对照检查一下你掌握了哪些核心知识：

1
for 的核心：遍历序列
for 变量 in 序列: 每次从序列中自动取出一个元素赋值给变量，取完就停，不需要手动维护计数器。
2
range() 的三种用法
range(stop) 从 0 到 stop-1； range(start, stop) 自定义起点； range(start, stop, step) 自定义步长。记住"左闭右开"：包含 start，不包含 stop。
3
嵌套 for：外层每转一圈，内层跑完一遍
总执行次数 = 外层次数 × 内层次数。常用于打印图案（矩形、三角形）、生成九九乘法表等二维结构。
4
break — 立刻退出整个循环
执行到 break，for 循环立刻结束，常用于"找到目标就停"。for 循环也支持 else 语句，循环正常完成（未被 break）时执行 else。
5
continue — 跳过本次，继续下一次
执行到 continue，跳过本次循环剩余代码，直接进入下一次遍历。常用于"过滤掉不需要的元素"。
🎯
for 循环的使用口诀 "知道遍历什么就用 for，range 控制数字范围，break 找到就跑，continue 不要就跳。"
